# BGM 트랙

이 폴더의 mp3 파일들은 **FFmpeg sine wave + 필터로 자체 합성**한 ambient pad 입니다.
저작권자 없음 → 상업적 사용 / 재배포 / 수정 모두 자유. 별도 attribution 불필요.

## 트랙별 분위기

| 파일 | 기본 음 | 분위기 | 어울리는 콘텐츠 |
|---|---|---|---|
| `ambient_warm.mp3`  | C3 기반 | 따뜻한 | 일상 연예 뉴스 |
| `ambient_calm.mp3`  | A2 기반 | 차분한·약간 어두운 | 이슈/심층 뉴스 |
| `ambient_light.mp3` | F3 기반 | 가볍고 밝은 | 가벼운 연예/스포츠 |

## 길이

각 30초. 영상이 더 길면 `make_video.py` 가 `-stream_loop -1` 로 무한 반복하고
`-shortest` 로 영상 길이에 맞춰 자릅니다. 페이드 인/아웃 트리트먼트는 mux 단계에서
다시 적용되므로 트랙 자체 페이드는 의미 없음.

## 재합성

분위기 마음에 안 들면 `make_video.py` 상단의 ffmpeg 명령을 참고해 다른 음/필터 조합으로
재생성. 예: lowpass `f=600`(어둡게) ↔ `f=1200`(밝게), tremolo `f=0.3`(느린 떨림) ↔ `f=0.8`(빠른 떨림).

## 왜 자체 합성?

1. **저작권 100% 안전** — Pixabay/Bensound 등도 라이선스 변경 리스크가 있음.
2. **외부 의존 없음** — 매 빌드마다 CDN 살아있는지 신경 안 써도 됨.
3. **파일 크기 작음** — 30s × 96kbps mono ≈ 360KB. 레포 부담 적음.
4. **Reels 알고리즘 충족** — 무음보다 어떤 오디오든 있는 게 점수 좋음.

순수 음원 퀄리티가 필요해지면(예: 멜로디 있는 BGM) 추후 라이선스 합법 트랙을 추가하면 됨.
