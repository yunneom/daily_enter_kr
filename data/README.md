# 쿠팡 파트너스 단축링크 설정

이 폴더의 `coupang_shortlinks.csv` 가 **실제 수익을 결정**합니다.
한 번 채우면 매 cron 게시마다 자동으로:
- 캡션에 `🍗 추천 야식·치킨 보러가기 → 프로필 링크 ⬇️` CTA 추가
- 자동 첫 댓글에 단축링크 추가 (24h 라스트클릭 쿠키 시드)
- `docs/index.html` 랜딩 페이지에 카테고리 카드 노출
- 공정위 디스클로저 자동 첨부

## 왜 이렇게 해야 하나

쿠팡 파트너스는 **자기 시스템에서 생성한 단축링크만 수수료를 인정**합니다.
`coupang.com/np/search?...&trcid=AF_ID` 같이 검색 URL에 AF ID 만 첨부하면
어트리뷰션이 안 잡혀 **수익 0원**. 따라서 단축링크를 미리 만들어 둬야 함.

OpenAPI Access/Secret 키가 있다면 딥링크 API 로 자동 생성 가능하지만,
현재 운영자는 AF ID 만 보유 → **수동 단축링크가 유일한 수익 경로**.

## 채우는 방법 (10-15분 소요)

1. https://partners.coupang.com 로그인
2. 좌측 메뉴 **"링크 생성"** 클릭
3. 아래 카테고리별 검색어로 검색 → 상품 또는 검색결과 페이지에서 "링크 생성"
4. **단축링크 (link.coupang.com/a/XXXXX)** 복사
5. `coupang_shortlinks.csv` 의 `shortlink` 컬럼에 붙여넣기
6. git commit + push → **다음 cron 게시부터 자동 반영**

## 카테고리 우선순위

조회수·관련도 기준 채우는 순서 권장 (먼저 채울수록 빨리 수익):

| 우선순위 | 카테고리 | 토픽 커버 | 검색어 힌트 |
|---|---|---|---|
| ⭐⭐⭐ | K-POP 굿즈 | 13 토픽 | 케이팝 응원봉 앨범 포카 슬리브 |
| ⭐⭐ | 야식·치킨 | 1 토픽 | 치킨 엽떡 닭발 안주 |
| ⭐⭐ | 홈트·요가 | 1 토픽 | 요가매트 홈트 덤벨 |
| ⭐⭐ | 직장인 가전 | 2 토픽 | 사무용 모니터암 키보드 마우스 |
| ⭐ | 그 외 14개 | 각 1 토픽 | CSV 참고 |

**K-POP 굿즈 1개만 채워도 매트릭스 28토픽 중 13토픽이 수익 활성됨.** 그것부터 시작.
