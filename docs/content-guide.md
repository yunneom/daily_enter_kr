# 릴스 콘텐츠 기획 가이드 (벤치마크)

> 외부 인플루언서 공유 자료 기반. 2주 클린 운영 후 분석할 때 적용 후보로 검토.

## 5가지 hook 유형

1. **후킹형** — 첫 2초 시선 잡기 (오해/실수 폭로)
2. **스토리형** — 경험·변화 (예전 나 → 깨달음 → 변화)
3. **How-To형** — 초보 가이드 (실전 순서)
4. **팁·체크리스트형** — 저장·공유 유도 (나열)
5. **댓글 유도형** — 자료 제공 약속으로 댓글 발화

## 우리 채널 적용성 평가

| 유형 | 우리 적용 | 이유 |
|---|---|---|
| 후킹형 (오해/실수) | ⭐⭐⭐ | "다들 1티어는 X라지만, 사실은..." 매트릭스와 잘 맞음 |
| 스토리형 | ❌ | faceless 자동화 — 인격체 톤 없음, 억지로 넣으면 가짜 |
| How-To형 | ❌ | 정보형 X, 놀이형 채널 |
| 팁·체크리스트형 | ✅ | 매트릭스 자체가 이미 체크리스트 |
| 댓글 유도형 (자료약속) | ⚠️ | DM 자동화 없음 → 약속 안 지키면 신뢰 깨짐. **댓글 미션은 OK** |

## 2주 후 분석 때 적용 검토 항목

### 1. `rule_hint` 회전 (hook 다양화)
지금: 토픽마다 1개 고정.
변경 후보: 토픽에 `rule_hints_rotation: [...]` 추가 → seed로 회전.

예: `girlgroup_4gen_tier1_10k`
```python
"rule_hints_rotation": [
    "각 포지션 1명씩 골라 합 1만원 — 진짜 1티어?",   # 체크리스트 (현재)
    "다들 4세대 1티어 알지만, 진짜 핵심은 이 9명",   # 오해형
    "1티어 픽에서 가장 많이 놓치는 멤버",            # 실수형
    "이 9명 다 알면 4세대 진짜 팬",                  # 도전형
    "당신의 1티어는 이중 누구?",                     # 질문형
]
```

### 2. 새 포맷 — "오해형 매트릭스"
- 일반 인식 vs 실제 BR 데이터 비교
- "사실 4세대 1티어는 X그룹이 아니다 (BR 출처)"
- BR 출처 이미 코드에 있음 → 자연 연결

### 3. 첫 카드(thumbnail) hook 글자 추가
지금 첫 프레임 = 매트릭스 풀샷 (정보 밀도 ↑, hook ↓)
변경 후보: 첫 0.5s만 큰 hook 문구 → 매트릭스 → 옵션 펼침

## 즉시 적용한 것

### auto_comment 댓글 미션화 (2026-06)
"댓글 키워드 → 자료" 형식은 자료 약속 못 지켜 신뢰 깨짐 → 대신
**"댓글로 답 적기 미션"** 형식으로 변경:
- 기존: "🤔 본인 픽 댓글로 ⬇️ 더 좋은 픽 있으면 알려주세요"
- 변경: "🔥 본인 픽 댓글로 ⬇️ '카리나'·'장원영' 중 1순위는?\n🔖 저장하고 친구한테 시키기"

→ 구체 선택지 제시로 댓글 시작 장벽 ↓, 저장 CTA 명시.

## 최종 체크리스트 (참고)

- 첫 2초 후킹이 있는가
- 타겟이 명확한가
- 구체적인 고민을 다루는가
- 저장할 가치가 있는가
- 댓글을 남길 이유가 있는가
- 자료 제공 CTA가 있는가 (우리는 약속 불가 → 스킵)
