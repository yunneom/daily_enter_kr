# 멤버 이미지 넣는 곳

이 폴더에 **순위 번호 파일명**으로 이미지를 넣으면 앱에 자동으로 표시됩니다.
없으면 그룹 색상 원형 카드(이름 표시)로 자동 폴백됩니다.

## 방법 1) 파일 드롭 (권장)
`web/public/members/{순위}.jpg` (또는 `.png` / `.webp`) — 예) `1.jpg` → 장원영

## 방법 2) URL 채우기
`web/data/member_images.json` 의 해당 순위 값에 이미지 URL 입력.
(위키미디어 커먼즈는 파일 페이지 URL만 줘도 자동 변환됩니다.)
